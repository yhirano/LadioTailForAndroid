# LadioTailForAndroid

Ladio TailはAndroid用の [ねとらじ](http://ladio.net/) 視聴ソフトウェアです。
Android端末からねとらじを視聴することができます。

## 主な機能

* 配信中のねとらじ番組一覧取得
* ねとらじ再生
